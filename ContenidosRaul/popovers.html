Componente

Popovers

Para que sirve

Los popovers en Bootstrap son componentes que se utilizan para mostrar información adicional sobre un elemento cuando el usuario interactúa con él, generalmente al hacer clic o pasar el ratón sobre dicho elemento. Son similares a los tooltips, pero con la capacidad de contener contenido más rico y estructurado, como encabezados, texto e incluso HTML.

Sus funcionalidades pueden ser:

  • Información Adicional:
  • Organización y Navegación
  • Mejora de la Experiencia del Usuario

Etiquetas HTML

Para habilitar ventanas emergentes con popovers se deben inicializar los elementos emergentes antes de poder usarlos. Una forma de inicializar todos los elementos emergentes de una página sería seleccionarlos por su data-bs-toggle atributo, de la siguiente manera:


            const popoverTriggerList = document.querySelectorAll('[data-bs-toggle="popover"]')
            const popoverList = [...popoverTriggerList].map(popoverTriggerEl => new bootstrap.Popover(popoverTriggerEl))

Los títulos se configuran mediante data-bs-title y el contenido del cuerpo mediante data-bs-content.


            data-bs-toggle="popover": Le dice a Bootstrap que el elemento asociado debe activar un popover cuando se interactúa con él.
            data-bs-title="Popover title": Proporciona el texto que se mostrará como el título del popover cuando se active.

Variantes

Existe una variante de Popovers llamados deshabilitados que se refieren a componentes de la interfaz de usuario que, aunque se presentan visualmente dentro del popover, están desactivados y no pueden ser interactuados por el usuario. Esto suele ser útil cuando se desea mostrar elementos que no están disponibles temporalmente o que requieren ciertas condiciones para ser activados.

Los elementos con el disabled atributo no son interactivos, lo que significa que los usuarios no pueden pasar el cursor sobre ellos ni hacer clic en ellos para activar una ventana emergente (o información sobre herramientas). Como solución alternativa, deberá activar la ventana emergente desde un contenedor <div> o <span>, idealmente y hacer que se pueda enfocar con el teclado usando tabindex="0".


                <span class="d-inline-block" tabindex="0" data-bs-toggle="popover" data-bs-trigger="hover focus" data-bs-content="Disabled popover">
                    <button class="btn btn-primary" type="button" disabled>Disabled button</button>
                  </span>

Ejemplo

En este ejemplo, podemos ver un código que utiliza la clase popovers de Bootstrap para desplegar mensajes emergentes sobre los botones. Cada botón genera un popover que se muestra en diferentes direcciones según se especifica en la línea de código correspondiente.

  • Botones: Cada botón está estilizado con Bootstrap y contiene un texto que describe la posición del popover.
  • Popovers: Son activados por los botones y muestran información adicional en diferentes posiciones (top, right, bottom, left).
  • Artributos de Datos:Utilizan data-bs-toggle, data-bs-placement, y data-bs-content para configurar los popovers.
