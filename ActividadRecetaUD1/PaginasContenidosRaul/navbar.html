Componente

Navbar

Para que sirve

La navbar o barra de navegación es un componente fundamental en el diseño de interfaces de usuario web. Se utiliza para crear una sección de navegación en una página web que generalmente contiene enlaces a otras partes del sitio, logotipos, formularios de búsqueda y otros elementos interactivos.

Pueden usarse para:

  • Crear Enlaces Rápidos: Contiene enlaces a las diferentes páginas o secciones del sitio web, permitiendo una navegación rápida y fácil.
  • Crear Menús Desplegables: Se Pueden incluir menús desplegables para organizar y mostrar enlaces adicionales.
  • Crear Identidad Visual: Ayudan a mantener una identidad visual coherente a través del sitio web.
  • Crear Formularios de Búsqueda: Puede incluir un campo de búsqueda para que los usuarios encuentren contenido específico en el sitio.

Etiquetas HTML


            <nav>: Define el contenedor principal de la barra de navegación.
            class="navbar bg-body-tertiary": Aplica las clases de Bootstrap a la barra de navegación.
            <nav class="navbar bg-body-tertiary">: Igual que en el primer bloque, define una barra de navegación con clases de Bootstrap.
            class="navbar navbar-expand-lg": Aplica estilos y funcionalidad de Bootstrap para una barra de navegación que se expande en pantallas grandes.
            bg-body-tertiary: Clase personalizada para el color de fondo.
            <a class="navbar-brand" href="#">: Enlace que representa el logotipo o nombre del sitio. Normalmente se encuentra en la esquina superior izquierda.
            <button class="navbar-toggler" type="button">: Botón que se utiliza para colapsar y expandir el contenido de la navbar en dispositivos pequeños.
            <ul class="navbar-nav me-auto mb-2 mb-lg-0">: Lista no ordenada que contiene los elementos de navegación.
            <li class="nav-item">: Elemento de la lista de navegación.
            <a class="nav-link" href="#">Link</a>: Enlace de navegación.
            <a class="nav-link disabled" aria-disabled="true">Disabled</a>: Enlace de navegación deshabilitado.

Variantes

Algunas de las variantes que exsiten en la clase navbar pueden ser muchas ya que en si un navbar e suna barra generalmente horizontal, en donde hay una serie de contenidos y como hemos visto pueden usarse varias etiquetas de HTML para combinarlo con esta clase.

No obstante existen varios tipos de navbar que se pueden crear como los siguientes:

Un navbar con formulario, que es específicamente un grupo de entrada (input-group). Es una barra de navegación simplificada que utiliza los componentes de Bootstrap para mostrar un campo de entrada dentro de la barra de navegación.


            <nav class="navbar bg-body-tertiary">
                <form class="container-fluid">
                  <div class="input-group">
                    <span class="input-group-text" id="basic-addon1">@</span>
                    <input type="text" class="form-control" placeholder="Username" aria-label="Username" aria-describedby="basic-addon1">
                  </div>
                </form>
              </nav>

Este otro código crea una barra de navegación simple que incluye un formulario con dos botones, uno principal y otro más pequeño. La barra de navegación utiliza clases de Bootstrap para el diseño y alineación, y los botones tienen estilos de Bootstrap que indican sus diferentes funciones y tamaños. Este tipo de navbar es útil para incluir elementos interactivos como botones de acción en la barra de navegación.


                <nav class="navbar bg-body-tertiary">
                    <form class="container-fluid justify-content-start">
                      <button class="btn btn-outline-success me-2" type="button">Main button</button>
                      <button class="btn btn-sm btn-outline-secondary" type="button">Smaller button</button>
                    </form>
                  </nav>

A veces, es posible utilizar el complemento de colapso para activar un elemento contenedor para el contenido que se encuentra estructuralmente fuera del .navbar. Como nuestro complemento funciona con la coincidencia id y data-bs-target, ¡eso se hace fácilmente!


                <div class="collapse" id="navbarToggleExternalContent" data-bs-theme="dark">
                    <div class="bg-dark p-4">
                      <h5 class="text-body-emphasis h4">Collapsed content</h5>
                      <span class="text-body-secondary">Toggleable via the navbar brand.</span>
                    </div>
                  </div>
                  <nav class="navbar navbar-dark bg-dark">
                    <div class="container-fluid">
                      <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarToggleExternalContent" aria-controls="navbarToggleExternalContent" aria-expanded="false" aria-label="Toggle navigation">
                        <span class="navbar-toggler-icon"></span>
                      </button>
                    </div>
                  </nav>

Este código crea una barra de navegación completa que incluye enlaces, menús desplegables, un formulario de búsqueda, y es responsiva para adaptarse a diferentes tamaños de pantalla utilizando las utilidades y clases de Bootstrap.

Esta explicación general da una vista de cómo los diferentes componentes y clases de Bootstrap trabajan juntos para crear una barra de navegación funcional y adaptable.

Esta barra de navegación incluye:

  • Un logotipo o nombre del sitio.
  • Un botón de colapso para pantallas pequeñas.
  • nlaces de navegación, incluido un menú desplegable.
  • Un formulario de búsqueda.>

Estas características hacen que la barra de navegación sea adaptable y fácil de usar en diferentes dispositivos y tamaños de pantalla.

                        <a class="nav-link" href="#">Link</a>
                      </li>
                      <li class="nav-item dropdown">
                        <a class="nav-link dropdown-toggle" href="#" role="button" data-bs-toggle="dropdown" aria-expanded="false">
                          Dropdown
                        </a>
                        <ul class="dropdown-menu">

                        </ul>
                      </li>
                      <li class="nav-item">
                        <a class="nav-link disabled" aria-disabled="true">Disabled</a>
                      </li>
                    </ul>
                    <form class="d-flex" role="search">
                      <input class="form-control me-2" type="search" placeholder="Search" aria-label="Search">
                      <button class="btn btn-outline-success" type="submit">Search</button>
                    </form>
                  </div>
                </div>
