Componente

Collapse

Para que sirve

Collapse es una clase de Bootstrap que se utiliza para crear elementos que pueden colapsarse o expandirse. Es decir, puedes ocultar o mostrar un contenido de manera dinámica mediante la adición o eliminación de la clase Collapse al elemento HTML.

La clase collapse en HTML es utilizada junto con Bootstrap para crear elementos que pueden expandirse y contraerse. Esta funcionalidad es especialmente útil para ocultar y mostrar contenido dinámicamente, haciendo que la interfaz de usuario sea más interactiva y limpia. Aquí hay algunos usos comunes de la clase collapse:

  • Mostrar Contenido
  • Organización y Navegación
  • Mejora de la Experiencia del Usuario

Etiquetas HTML


            data-bs-toggle="collapse": Indica que este botón se usará para activar el colapso.
            data-bs-target="#collapseWidthExample": Especifica el id del contenedor que será colapsado o expandido.
            aria-controls="collapseWidthExample": Relaciona el botón con el contenido colapsable.
            class="collapse collapse-horizontal": Clase de Bootstrap que permite el colapso horizontal del contenido.
            id="collapseWidthExample"
            aria-expanded="false": Atributo de accesibilidad que indica el estado inicial (colapsado).

Atributos

  • .collapse oculta el contenido
  • .collapsing Se aplica durante las transiciones
  • .collapse.show Muestra el contenido

En general, recomendamos usar a <button> con el data-bs-targetatributo. Si bien no se recomienda desde un punto de vista semántico, también puede usar un enlace con el hrefatributo (y a role="button"). En ambos casos, data-bs-toggle="collapse"se requiere the.

Variantes

El complemento de colapso admite el colapso horizontal. Agregue la .collapse-horizontalclase modificadora para realizar la transición widthen lugar de heighty establezca un widthen el elemento secundario inmediato. Siéntase libre de escribir su propio Sass personalizado, usar estilos en línea o usar nuestras utilidades de ancho .


            <p><button class="btn btn-primary" type="button" data-bs-toggle="collapse" data-bs-target="#collapseWidthExample" aria-expanded="false" aria-controls="collapseWidthExample">
                  Toggle width collapse
                </button>
            </p>
            <div style="min-height: 120px;">
            <div class="collapse collapse-horizontal" id="collapseWidthExample">
                <div class="card card-body" style="width: 300px;">
                    This is some placeholder content for a horizontal collapse. It's hidden by default and shown when triggered.
                </div>
            </div>
            </div>

Ejemplo

  • Inicialmente, el contenido dentro de div class="collapse"> está oculto.
  • Cuando se hace clic en el enlace o botón, el contenido se despliega (o colapsa) debido a los atributos data-bs-toggle y data-bs-target/href que enlazan al id del contenido (collapseExample).
  • Accesibilidad: Los atributos aria-expanded y aria-controls ayudan a los lectores de pantalla a entender la relación entre los botones y el contenido colapsable.

Este sencillo ejemplo muestra cómo puedes implementar colapsos de contenido en tu página web utilizando Bootstrap. Puedes adaptar esta estructura para diferentes propósitos, como menús desplegables, paneles de información, etc.
